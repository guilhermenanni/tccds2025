
 DraftMe – Plataforma de Oportunidades Esportivas

O DraftMe é um projeto de TCC composto por um aplicativo mobile e um site, desenvolvido para conectar jovens atletas a oportunidades no mercado esportivo.
A plataforma possibilita que organizadores publiquem seletivas, enquanto atletas podem visualizar, se inscrever e acompanhar suas participações.

 Funcionalidades:

   Atletas:
  -  Cadastro e login
  -  Perfil esportivo
  -  Visualização de seletivas
  -  Inscrição nas seletivas
  -  Acompanhamento das inscrições

   Organizadores:
  -  Criação de seletivas
  -  Edição e exclusão
  -  Visualização dos inscritos

 Tecnologias Utilizadas:

   Mobile:
  - React Native
  - Expo
  - TypeScript
  - React Navigation
  - Axios

   Site:
  - HTML
  - CSS
  - JavaScript
  - PHP

   Backend:
  - PHP
  - MySQL

 Arquitetura do Sistema:

Mobile (React Native)
Site (HTML/CSS/JS)
        ↕
     Backend (PHP)
        ↕
       MySQL
